# Overview

`grounded-research` is a Tyler V1-oriented grounded research pipeline. The live
system decomposes a question, gathers evidence, runs analyst passes, extracts and
adjudicates claims, resolves decision-critical disputes, and exports a traceable
report.

The maintainer problem is not lack of documentation. The problem is too many
historical documents with different authority levels. The current organization
keeps active status in the Tyler ledger/status docs, moves completed execution
plans to archive, and uses this wiki for durable concept navigation.
